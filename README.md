📊 Análise Estatística do Dataset Tips
📌 Descrição do Projeto

Este projeto tem como objetivo aplicar técnicas de pré-processamento e estatística descritiva utilizando o dataset Tips, disponível no Kaggle.

A proposta é analisar os dados de gorjetas em um restaurante, extraindo informações relevantes por meio de medidas estatísticas.

📂 Dataset Utilizado

Fonte: Kaggle

Link: https://www.kaggle.com/datasets/sakshisatre/tips-dataset

Descrição: Contém informações sobre contas de clientes, incluindo valor total, gorjeta, sexo, fumante ou não, dia da semana, período e número de pessoas.
